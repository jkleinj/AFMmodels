AFM Models

see also:
Malaria Parasite Actin Polymerization and Filament Structure
<http://dx.doi.org/10.1074/jbc.M110.142638>

* rabbit model_rJ:  --rotAngle 166.54 --zOffset 27.8 --rAxis 15.8 --nModel 80
